## Pir紅外線為選配品，若條件允許且需要精準快速開燈，建議選購Pir紅外線感測器，最多支持擴充2個。
二代毫米波感應器已經升級為LD2412，增加了自動背景偵測的功能，現場在沒有wifi環境下可以先將感測器固定好，橋好角度按一下面板前的小洞會開始計時10秒(人遠離淨空現場)，短絀音結束後開始偵測背景， 約莫1分鐘當第二次短絀音出現後級偵測成功。
