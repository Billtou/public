# 1.1 接入自家網路(WIFI)
## -  上電稍待超過30秒手機靠近設備找wifi熱點 automate_gd3-xxxxxx，接入後手機會自動跳出設訂wifi頁面，指定好自家wifi與密碼即可。
# 1.2 接入自家網路(網線)
## -  上電插上網線即可。
### 注意: 若已接入自家wifi但要切換為網線連接，請依 "重置設備方式" 即可

# 備註: 防夾自動化說明
## 當按下關門(下)的動作會送電給雷射開啟偵測模式，直到門磁感應反饋關閉為止，期間若有異物阻擋就會停止關門動作，畫面會出現有障礙物的訊號，可於障礙物排除後繼續按關閉或按開門將門打開方便移除障礙物。
