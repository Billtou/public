# AUTOMATE GD3 Homekit 版本說明
## 重置設備方式(清除wifi)
![081733](/GD_3_homekit/image/image13.JPG)
# 1.1 接入自家網路(WIFI)
## -  上電稍待超過30秒手機靠近設備找wifi熱點 automate_gd3-xxxxxx，接入後手機會自動跳出設定wifi頁面，指定好自家wifi與密碼即可。
# 1.2 接入自家網路(網線)
## -  上電插上網線即可。
### 注意: 系統開機會先搜尋網線30秒，找不到網線才會跳出AP這時即可接入WIFI，若已接入wifi但要切換為網線連接，請接入網線後依上述 "重置設備方式" 清除wifi即可。
# 1.3 Apple home掃描QR-Code接入蘋果家庭app 或輸入引導號碼 11188999 即可。

# 備註: 防夾自動化說明
## 當按下關門(下)的動作會啟動雷射偵測模式，直到門磁感應反饋關閉為止，期間若有異物阻擋就會停止關門動作，畫面會出現有障礙物的訊號，可於障礙物排除後繼續按關閉或按開門將門打開方便移除障礙物。
