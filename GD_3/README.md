# 接入自家網路(WIFI)
## -  上電稍待1~2分鐘手機靠近設備找wifi熱點 automate_gd3-xxxxxx，接入後手機切換到瀏覽器輸入192.168.4.1即可指定自家wifi與密碼，HA會自動找到設備。
# 接入自家網路(網線)
## -  上電插上網線即可。

# 備註: 防夾自動化說明
## 當按下關門(下)的動作會送電給雷射開啟偵測模式，直到門磁感應反饋關閉為止，期間若有異物阻擋就會停止並在後台的 Emergency Stop 開關會有點動的訊號，後期可以自行使用該訊號作為自動化的觸發機制。
