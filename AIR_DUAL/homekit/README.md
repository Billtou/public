## 1. 產品簡介

AirDual 是 AUTOMATE 推出的 HomeKit 智能除濕清淨二合一控制器,**直接接入 Apple Home**,不需專屬 Hub 或雲端帳號。

二合一機本體沿用 TaiSEIA 標準通訊協定,模組透過 UART 跟原廠 MCU 通訊。Apple Home 上會看到**兩台子裝置**:
- 「除濕機」磚塊 — 可看目前濕度 / 調整目標濕度
- 「空氣清淨機」磚塊 — 可看空氣品質燈號(優/良/普通/差/極差)+ PM2.5 讀數

模式切換(自動除濕 / 連續 / 衣物乾燥 / 空氣清淨 / 防霉…)由 Apple Home 或實體面板,Web UI 純粹**顯示**目前運轉狀態。

> 跟 Matter 產品不同,**HomeKit 是 Apple 專屬生態**,只能接入 Apple Home(iPhone / iPad / Mac),不能加進 Google Home / Home Assistant / Alexa。如需跨生態,請另選 Matter 版本產品。

### 1.1 支援機型

依出廠 profile 區分,本控制器支援以下二合一機:

| Profile | 廠牌 | 類型 | 代表機型 |
|---|---|---|---|
| 0 | 國際牌 Panasonic | 除濕 + 空氣清淨二合一 | F-YZUX 系列 |
| 1 | 日立 Hitachi | 除濕 + 空氣清淨二合一 | RD-200FQ / RD-240HQ |
| 2 | 日立 Hitachi | 全熱交換器(ERV) | (未來推出) |

> Profile 出廠燒入,**終端使用者不需設定**。包裝盒側標籤會註明機型。

### 1.2 規格速覽

| 項目 | 內容 |
|---|---|
| 通訊協定 | HomeKit (HAP) over Wi-Fi |
| Wi-Fi | 2.4 GHz 802.11 b/g/n(**不支援 5 GHz**)|
| BLE | 不需要(HAP 走 Wi-Fi 配對)|
| 處理器 | ESP32-C3(RISC-V 32-bit, 160 MHz, 4 MB Flash)|
| MCU 介面 | UART1 9600 8N1(TaiSEIA 標準) |
| Apple Home 對映 | HumidifierDehumidifier + AirPurifier + AirQualitySensor |
| 感測回報 | 室溫 / 室內濕度 / PM2.5 / 空氣品質 / 水箱 / 濾網 |
| 工作環境 | 0–40 °C / 10–90% RH(無凝結)|

### 1.3 包裝內容

- AirDual 控制器模組 × 1(已預先裝在二合一機的 Wi-Fi 模組槽位)
- 快速入門卡(含 HomeKit QR Code + 8 碼配對碼)× 1
- 原廠遙控器 / 面板(隨二合一機出貨,非本產品內容物)

> 模組已在出廠時裝入機台,**終端使用者不需要拆機**。

## 2. 硬體說明

### 2.1 安裝與接口

二合一機本體接 AC 110V,**無需額外接線**。AirDual 模組接在機台的 Wi-Fi 模組槽位,透過 UART 跟原廠 MCU 通訊,Wi-Fi 天線內建。

- **內建 Wi-Fi 天線**:無需外接
- **原廠面板 / 遙控**:OEM 原有,獨立於 AirDual 模組
- **狀態 LED**(模組上,藍光):見章節 3.1
- **重置鍵**(模組上,小孔):見章節 3.2

### 2.2 配對碼資訊

| 項目 | 數值 |
|---|---|
| HomeKit QR Code | 快速入門卡 / 機身底部標籤 |
| 8 碼配對碼 | 格式 `XXX-XX-XXX`(每片唯一,出廠燒入)|
| Manufacturer | `AUTOMATE` |
| Model | `AirDual-TS-HAP` |
| 預設裝置名稱 | `AirDual-XXXXXX`(後 6 碼為 Wi-Fi MAC)|
| 預設 hostname | `automate-airdual-XXXXXX.local` |

> 配對碼遺失時,可在 Web UI(章節 6)「裝置資訊」區查看 hostname 反查機身;或從機身底部標籤掃 QR。

## 3. 指示燈與按鍵行為

### 3.1 狀態 LED

模組上的單色藍燈會依優先序顯示連線狀態:

| 條件 | LED 行為 | 意義 |
|---|---|---|
| 重置警告(已長按 ≥ 5s) | **急閃** 100ms 亮 / 100ms 滅 | 「再按 5 秒就觸發工廠重置」警告 |
| Wi-Fi 未連線 | **慢閃** 0.2s 亮 / 1.8s 滅(每 2 秒一次) | 配網模式,需配對或 Wi-Fi 重連 |
| HomeKit 未配對 / 機台 UART 未連 | **雙閃** 2 秒內閃兩下 | Wi-Fi 通了但 HomeKit 還沒加入 / 機台沒通電 |
| 正常運轉 | **熄滅** | 三條都通(Wi-Fi + HomeKit + 機台),不擾人 |

> 規範:**LED 預設不打擾**。閒置正常時不亮,有問題才亮。

### 3.2 重置按鈕

| 按法 | 時間 | 結果 |
|---|---|---|
| 短按 | < 0.6 秒 | 忽略(debounce) |
| 單擊 | 0.6 – 5 秒 | 切換主電源(等於遙控的 ON/OFF) |
| 長按 | 5 – 10 秒 | 進入**重置警告**(LED 急閃),放手取消 |
| 長按 | ≥ 10 秒 | **工廠重置**:清除 HomeKit 配對 + Wi-Fi 設定 + 重啟 |

### 3.3 原廠面板 / 遙控

原廠面板 / 遙控走廠商既有通路,**獨立於 AirDual 模組**:
- 不受 Wi-Fi 斷線影響
- 操作觸發的狀態變化,Apple Home 端 1 秒內同步看到
- 反之,Apple Home 控制也會反映在實體機台,**原廠面板顯示也會同步**

## 4. 接入 Apple Home(初次配對)

### 4.1 配對前確認

- [ ] 路由器 **Wi-Fi 2.4 GHz** 可用(AirDual 不支援 5 GHz)
- [ ] iPhone / iPad **已加入「家庭」App**(iOS 16.1+)
- [ ] 至少有一台 **HomeKit 家居中樞** 在線:
  - HomePod / HomePod mini / Apple TV(4K)/ iPad(留在家裡作 Hub 角色)
- [ ] 手機跟二合一機距離 < 1 公尺

### 4.2 配對流程

1. 通電二合一機,等模組完成上電(約 5 秒,LED 進入慢閃配對模式)
2. iPhone 開「**家庭**」App → 右上角 **+** → **加入或掃描配件**
3. 對準 QR Code 掃描;或選「**沒有代碼/無法掃描**」→ 手動輸入 8 碼 setup code
4. iPhone 自動傳 Wi-Fi 憑證給模組
5. 配對完成 → 出現 **「AirDual-XXXXXX」**(可在 App 內改名)
6. 預設會看到**兩台子裝置**(磚塊):
   - 除濕機(濕度滑桿)
   - 空氣清淨機(空氣品質燈號 + PM2.5 讀數)

### 4.3 配對後預期

- 主屏幕磚塊:可直接看 / 控
- 詳細頁(長按磚塊):可看完整資訊(室溫、目前濕度、空氣品質、PM2.5、水箱、濾網)
- 自動化(Automation):可設定例如「PM2.5 > 35 → 開空氣清淨機」、「家裡有人 → 開除濕」這類條件式規則

## 5. Apple Home 控制

### 5.1 除濕機磚塊

| 元件 | 功能 |
|---|---|
| **磚塊點擊** | 切換除濕機 ON / OFF |
| **濕度滑桿** | 拖動 = 設定目標濕度(30–70 %)|
| **詳細頁** | 顯示「目前濕度」+「目標濕度」+「室溫」 |

### 5.2 空氣清淨機磚塊

| 元件 | 功能 |
|---|---|
| **磚塊點擊** | 切換空氣清淨 ON / OFF |
| **燈號顯示** | 綠 / 黃 / 橘 / 紅(對應 優 / 良 / 普通 / 差 / 極差) |
| **詳細頁** | 顯示「空氣品質」等級 + 「PM2.5」數值(µg/m³) |

### 5.3 自動化建議

Apple Home → 自動化 → 「當…時」+「執行…」:

- **PM2.5 > 35 µg/m³ → 開空氣清淨機**(空污超標自動清淨)
- **目前濕度 > 70% → 開除濕機**(濕度爆表自動除濕)
- **家裡有人離開 → 兩台都關**(節能)
- **晚上 22:00 → 切到「保持乾燥」模式**(夜間維持)

## 6. Web UI:首頁(狀態顯示)

### 6.1 連線方式

配對成功後,AirDual 在區網提供 Web UI。打開瀏覽器,網址兩種擇一:

- **直接 IP**:在家庭 App → 點裝置磚塊 → 設定 → 看「Wi-Fi 網路位址」(例 `http://192.168.1.123`)
- **主機名稱**:`http://automate-airdual-XXXXXX.local`(XXXXXX 為機身 Wi-Fi MAC 後 6 碼)

> 手機與 AirDual 必須在**同一個 Wi-Fi 網段**。
>
> 部分路由器(Asahi 預設 / 美國 ISP Comcast 等)開「Guest 隔離」,要關掉才能訪問。

### 6.2 裝置資訊

| 欄位 | 內容 |
|---|---|
| **韌體版本** | 例 `v0.1.1` |
| **可用記憶體** | 例 `105kB`(< 30kB 時轉紅警告)|
| **運行時間** | 例 `30m 30s` / `2h 15m` |
| **通訊狀態** | `HomeKit 已配對 (192.168.1.x)` + `正常 (wifi)` |
| **製造日期** | `YYYY-MM`,出廠燒入 |
| **使用說明** | 點藍色連結即跳到本份 README |

### 6.3 控制區與狀態區

Web UI 主要為**狀態顯示**,大部分控制建議透過 Apple Home App 或實體面板。卡片內分兩塊:

**控制區**(可動,僅 2 項):

| 元件 | 功能 |
|---|---|
| **nanoeX 離子 toggle** | 即時切換離子發生器(獨立硬體開關,不依 mode)|
| **蜂鳴 toggle** | 即時切換按鍵 / 警示音 |

**唯讀狀態區**:

| 欄位 | 顯示 |
|---|---|
| 電源 | 運轉中 / 已關機 |
| 目前濕度 | 例 `64 %`(無感測器讀數時顯示 `--`) |
| 室溫 | 例 `24.3 °C`(無感測器讀數時顯示 `--`) |
| 目標濕度 | 例 `55 %` |
| 模式 | 自動除濕 / 保持乾燥 / 連續除濕 / 衣物乾燥 / 空氣清淨 / 防霉抑菌 |
| 風量 | 自動 / 靜音 / 微風 / 標準 / 強風 / 急速 |
| 擺風 | 運轉中 / 停止 |
| 水箱 | 正常 / 已滿(紅色警示)|
| 濾網 | 正常 / 需更換(橘色警示)|
| 空氣品質 | 優 / 良 / 普通 / 差 / 極差 |
| PM2.5 | 例 `12 µg/m³` |

### 6.4 感測讀數說明

- 顯示 `--` 或 `— µg/m³` = 機台目前**沒回報**該感測值(可能機型不支援、感測器離線、或機台剛開機尚未 polling 一輪完成)
- 室溫顯示 `-- °C` 通常代表機台尚未通電 / UART 斷線
- 全熱交換(Profile 2)機型:**沒有水箱 / 濾網 / 空氣品質 / PM2.5** 欄位,而是顯示「戶外溫度 / CO2 / PM2.5」

## 7. 模式與功能

### 7.1 運轉模式

由 Apple Home 或原廠面板切換(Web UI 不提供模式控制):

| 模式 | 用途 |
|---|---|
| **自動除濕** | 機台依當前濕度自動決定壓縮機 ON/OFF,維持在目標濕度附近 |
| **保持乾燥** | 比自動更積極,維持較低濕度(衣櫃 / 倉庫場景) |
| **連續除濕** | 不看目標濕度,壓縮機持續運轉至水箱滿 |
| **衣物乾燥** | 風扇強勁 + 持續除濕,適合室內曬衣 |
| **空氣清淨** | 壓縮機停,只跑風扇 + 濾網 + 離子,純空氣淨化 |
| **防霉抑菌** | 機台依「防霉演算法」自動間歇除濕 + 開離子,目的防霉 |

### 7.2 風量段位

由 Apple Home 風扇磚塊(滑桿)或原廠面板切換:

| 風量 | 對應 Apple Home 百分比 |
|---|---|
| 自動 | 0% – 16% |
| 靜音 | 17% – 33% |
| 微風 | 34% – 50% |
| 標準 | 51% – 66% |
| 強風 | 67% – 83% |
| 急速 | 84% – 100% |

> Apple Home 滑桿任意中間值會自動量化到最近的物理段位。

### 7.3 nanoeX 離子

**獨立硬體開關**,不依 mode 自動切換:

- **打開後**:離子發生器持續運轉,不論機台是除濕、空清、防霉、衣物乾燥都會跑
- **關閉後**:離子發生器停,即使切到防霉 mode 也不會自動開
- **整機 OFF**:離子物理上停(沒電),但 state 記得;再開機通常自動恢復(看廠牌韌體)

### 7.4 水箱與濾網

| 顯示 | 處理 |
|---|---|
| 水箱「正常」 | 不用管 |
| 水箱「**已滿**」(紅 pill) | 倒水後重新插入,機台自動繼續運轉 |
| 濾網「正常」 | 不用管 |
| 濾網「**需更換**」(橘 pill) | 拆下清洗 / 更換濾網。完成後依機台說明書按重置鍵清除提示 |

## 8. OTA 韌體更新

### 8.1 更新流程

1. 連線 Web UI → 底部 **版更** Tab
2. 系統自動向官方檢查最新版本
3. 若有新版,顯示「**有新版本可更新**」+「**開始更新**」按鈕
4. 點擊「**開始更新**」→ 確認對話框 → 自動下載
5. 下載完成自動重啟,Web UI 自動 reload 回首頁

整個過程約 1–2 分鐘,期間 HomeKit 短暫離線(實體機台不受影響,原廠面板 / 遙控仍可用)。

### 8.2 更新會保留 / 清除什麼

| 項目 | 是否保留 |
|---|---|
| HomeKit 配對(Apple Home 加入狀態)| ✅ 保留 |
| Wi-Fi 帳密 | ✅ 保留 |
| 防複製綁定 | ✅ 保留 |
| 出廠製造日期 | ✅ 保留 |
| 機型設定(Profile)| ✅ 保留 |
| 韌體版本 | ❌ 換新版 |

## 9. 工廠重置

### 9.1 硬體重置

- 按住模組重置鍵 **5 秒**:進入警告(LED 急閃),放手取消
- 持續按到 **10 秒以上**:**工廠重置觸發**,清除 HomeKit 配對 + Wi-Fi 設定,自動重啟

### 9.2 Web UI 重置

- 版更頁 → 「重新配對裝置」card → **「清除配對」** 紅色按鈕
- 確認對話框 → 自動重啟

> 此項是 **HomeKit-only reset** — 只清配對 + Wi-Fi。等同硬體長按 10 秒。

### 9.3 重置會清除什麼

| 項目 | 是否清除 |
|---|---|
| HomeKit 配對 | ✅ 清除 |
| Wi-Fi 帳密 | ✅ 清除 |
| 防複製綁定 | ❌ 保留(只有完整 erase_flash 才清)|
| 出廠製造日期 | ❌ 保留 |
| 機型設定(Profile)| ❌ 保留 |
| 韌體版本 | ❌ 不變 |
| 裝置認證資料 / Setup Code | ❌ 不變 |

> 重置後跟全新開封一樣,需重新配對。家庭 App 會看到「附近」分區自動出現此裝置(裝置會自動 mDNS 廣播)。

## 10. 故障排除

### 10.1 配對相關

| 現象 | 原因 | 處理 |
|---|---|---|
| 家庭 App 找不到裝置 | 距離太遠 / Wi-Fi 是 5 GHz | 確認距離 < 1 m、Wi-Fi 走 2.4 GHz |
| 顯示「無法加入配件」| 無 Home Hub / 配對碼錯誤 | 確認家中有 HomePod / Apple TV / iPad 作 Hub;重新掃 QR |
| 配對中途斷開 | 路由器訊號弱 / 模組電源不穩 | 移近路由器再試;檢查機台 AC 電源是否穩定 |
| 配對成功但顯示「無回應」| DNS / mDNS 問題 / 不同 SSID | 確認手機跟模組同一個 2.4 GHz SSID,路由器關「Guest 隔離」|

### 10.2 運作相關

| 現象 | 原因 | 處理 |
|---|---|---|
| Apple Home 顯示裝置離線 | Wi-Fi 斷線 / 路由器重啟 | 等 1–2 分鐘自動回連;不行就重新通電 |
| 拉目標濕度滑桿但機台沒反應 | UART 通訊異常 / 機台關機 | 確認機台電源 ON;重啟模組 |
| Web UI 數值顯示 `—` | 機台沒回報該感測值 | 部分機型不支援、或機台剛開機 polling 還沒一輪 |
| 原廠面板可動,Apple Home 沒同步 | UART 通訊異常 | 重啟模組 / 等 1 秒內 polling 自動同步 |
| 切換模式 1–2 秒後 Apple Home 又跳回 | Apple Home 本地緩存 | 屬正常,放著等下一輪 polling 對齊 |

### 10.3 感測讀數異常

| 現象 | 原因 | 處理 |
|---|---|---|
| 室溫顯示「-- °C」 | 感測器離線 / UART 沒通 | 重啟機台 + 模組 |
| PM2.5 一直 `—` | 機型不支援(只有入門款)/ 模組 Profile 設錯 | 確認包裝盒側標籤的機型 |
| 空氣品質燈號永遠是 `—` | 機型只有除濕、無空清 | Apple Home 空氣清淨機磚塊不會出現空氣品質燈號(預期行為)|
| 濕度數值跟其他濕度計差很多 | 機台感測器位置 / 校正 | 與原廠儀器比對,大誤差請聯絡廠商換感測器 |

### 10.4 OTA 相關

| 現象 | 原因 | 處理 |
|---|---|---|
| 版更頁顯示「**無法取得版本**」| 裝置連不到 GitHub | 檢查網路,可能是 DNS / 防火牆問題 |
| 「開始更新」後一直 0% | 下載卡住 | 等 5 分鐘;不行就重啟裝置再試 |
| 更新失敗 | 韌體損毀 / 供電不足 | 重啟裝置(會走舊版),確認供電後重試 |

## 11. 安全使用

- **AC 110V 高壓側由二合一機原廠處理**,模組本身為低壓 5V 工作
- 接線 / 拆卸**先斷電**
- 模組已預先裝入機台,**請勿自行拆換或改裝**
- 工作環境溫度 0–40 °C,**勿讓控制盒被悶熱包圍**
- **戶外不可使用**,完全曝雨環境會損壞
- 模組內含無線電路,改裝會違反電信法規
