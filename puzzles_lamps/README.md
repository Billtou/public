## 最後燈座安裝提醒

### 有孔的零件最後安裝，在安裝到最後1片的時候先把燈泡鎖上燈座放入燈罩中，電線穿過孔洞鎖上帽套即可，**注意勿所過緊能固定不搖晃即可**。
