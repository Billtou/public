# 伸縮式軌道窗簾使用說明

## 接入自家wifi

  上電後手機搜尋設備發出的AP熱點(cb3s-curtain xxxxxx) 點選接入這個名稱的wifi後會自動跳轉到指定自家wifi的頁面，輸入後儲存即可，若沒跳轉到指定頁面，手機切換到瀏覽器輸入192.168.4.1 即可依序填入自家wifi資訊HA就會自動發現。

## 首次使用或斷電後首次使用

  插電後馬達首次運轉時會自動開合一遍測量窗簾行程，然後記憶直到斷電
