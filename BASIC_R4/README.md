# AUTOMATE Sonoff Basic R4 Matter 智慧開關使用手冊

歡迎使用 AUTOMATE Sonoff Basic R4 Matter 智慧開關！本設備採用最新 Matter 智慧家庭標準協定，讓您能輕鬆整合至 Apple Home、Google Home、Amazon Alexa、SmartThings 及 Home Assistant 等支援 Matter 的平台中，實現無縫的跨平台智慧家庭體驗。

## 1. 產品簡介

AUTOMATE Sonoff Basic R4 是一款支援 Matter-over-WiFi 的智慧開關改裝模組。透過本設備，您可以將傳統家電或燈具升級為智慧設備。擁有實體按鈕控制、網頁面板控制，更支援「Magic Switch（魔術開關）」功能，完美相容並保留您家中的傳統牆壁開關控制習慣。

## 2. 產品規格

*   **產品設備名稱**: AUTOMATE SONOFF-R4(matter)
*   **輸入電源**: 100–240 V AC (50/60Hz)
*   **最大負載額定值**: 10 A / 250 VAC
*   **網路通訊協定**: Wi-Fi (2.4GHz), Matter over Wi-Fi
*   **斷電記憶**: 支援，復電後設備將自動恢復斷電前繼電器的狀態 (Restore Previous State)。

## 3. 接線與安裝安全提示

> **⚠️ 警告：觸電危險**
> 安裝與接線前，**請務必關閉總電源無熔絲開關 (Breaker) 或拔除電源！** 強烈建議由具備相關資格的專業電工技師進行安裝。

*   **Input (電源輸入端)**：請將市電的火線 (L) 與零線 (N) 分別接入設備上的 `L In` 與 `N In` 端子。
*   **Output (電源輸出端)**：請將連接至燈具或負載設備的火線與零線接至 `L Out` 與 `N Out` 端子。
*   **Magic Switch 功能**：若要保留傳統牆壁開關功能，請依照設備接線圖，將外部開關正確接入指定的端子孔位（請勿帶電接入外部電壓）。

## 4. 首次設定與 Matter 配對

開機後若尚未進行過配對，設備上的藍色狀態指示燈將以 **1 秒閃爍一次 (1 Hz)** 的頻率慢閃，代表已啟動並準備好進行 Matter 配對。

**配對步驟：**
1.  請將您的智慧型手機連線至家中的 **2.4GHz Wi-Fi** 網路。
2.  打開您所使用的智慧家庭 App（如 Apple Home 或 Google Home）。
3.  選擇「加入配件」或「新增裝置」，並選擇加入「Matter 裝置」。
4.  當系統要求掃描 QR Code 但因故無法掃描，或需要手動輸入憑證時，請選擇 **「使用設定碼配對」**(或「沒有 QR Code」) 的選項。
5.  輸入本設備固定配對碼：
    ### `11188999`
6.  依畫面指示等待設備連接 Wi-Fi 並註冊到您的智慧家庭系統。配對成功後，指示燈將跟隨開關目前的開/關狀態顯示。

## 5. 控制與操作方式

本設備支援多元的控制方式，且無論您透過哪種方式控制，設備狀態皆會快速同步至所有的 Matter 平台：

1.  **智慧家庭 App 遠端控制**：透過 Apple Home 或 Google Home 等 App 隨時隨地監控與遙控開關。
2.  **設備實體按鈕**：單擊設備外殼上的黑色小按鈕，即可立即切換開關通電狀態。
3.  **傳統牆壁開關 (Magic Switch)**：若您有接上傳統翹板牆壁開關，只要撥動實體開關，偵測到切換就會同步觸發設備狀態改變。*(此功能預設啟用，若不需要可登入網頁介面關閉)*。
4.  **區域網路網頁控制 (Web UI)**：
    *   在瀏覽器輸入設備的 IP 位址（例如：`http://192.168.x.x`）即可進入設備的網頁主控台。
    *   網頁版提供大圓形按鈕供直覺觸控操作，亦可在此設定與檢視設備狀態。

## 6. LED 指示燈狀態說明

透過設備上的藍色 LED 指示燈，可以立刻辨識目前的運作狀態：

| LED 狀態 | 代表意義 |
| :--- | :--- |
| **慢閃 (每秒 1 次)** | **待配對狀態 (Ready to pair)**，等待加入 Matter 網路。 |
| **熄滅** | 已連線，且開關處於 **關閉 (OFF)** 狀態。 |
| **常亮** | 已連線，且開關處於 **開啟 (ON)** 狀態。 |
| **快閃維持 3 秒** | **裝置識別 (Identify)**，當您在 App 內點選「識別設備」時觸發，方便您尋找設備。 |
| **極快閃 (每秒 10 次)** | **準備重設 (Reset Pending)**，按鈕已長按超過 5 秒，繼續按住不放將觸發恢復原廠設定。 |

## 7. 設備重設與多平台綁定

若您更換了 Wi-Fi 路由器，或希望清除設備資料重新綁定網路，請使用設備身上的實體按鈕操作：

*   **重新開放配對模式 (打開 Commissioning Window)**：
    *   **操作方式**：快速**雙擊**實體按鈕（兩次點擊間隔需小於 0.6 秒）。
    *   **功能說明**：若設備已在網路中，但您想再將設備加入另一個平台（如已在 Apple Home，想加入 Google Home），除了從 App 中產生新的配對碼以外，也可以使用此方式手動喚醒 Matter 配對廣播訊號。
*   **恢復原廠設定 (Factory Reset)**：
    *   **操作方式**：**長按**實體按鈕 **10 秒以上**。
    *   **過程說明**：長按至 5 秒時，指示燈會開始極快閃爍；此時請不要鬆開，繼續按住直到大約 10 秒設備強制重啟。
    *   **功能說明**：所有的 Wi-Fi 設定與 Matter 配對紀錄將全數清除不留痕跡，重啟後藍燈慢閃，供您重新配對。

## 8. Web 介面與韌體更新

本機內建有網頁伺服器，只要在同一區域網路內，利用網頁瀏覽器存取設備 IP 路徑 `http://<設備IP>/update` ，即可進入 **「更新與進階設定」** 頁面：

*   **雲端韌體更新 (OTA)**：您可以在此檢查 GitHub 上是否有最新韌體版本並執行線上更新，確保設備安全性與獲得最新功能。
*   **清除系統記憶體**：提供軟體等級的「恢復原廠設定」選項 (NVS Clear)，按下後將清除資料並重啟。
