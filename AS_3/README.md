## 1. 產品簡介

AS-P3 是 AUTOMATE 推出的**智能空間控制器**,適合廁所、浴室、陽台、玄關、儲藏室等小空間。它整合**人體偵測、光照、溫濕度**感測,自動控制**電燈**與**風扇 / 抽風機**兩路:

- **有人自動開燈**:偵測到人且環境夠暗時自動開燈,人離開延遲後自動關。
- **有人自動通風**:偵測到人自動開風扇,人離開倒數後關。
- **濕度自動除濕**:濕度超過設定值自動開風扇排濕,降回門檻自動停。
- **保留實體開關**:牆面開關照常可手動控制,並可選擇與自動邏輯「解耦」。

採 **ESPHome** 韌體,透過 Wi-Fi 接入 **Home Assistant**,所有功能與參數都在 Home Assistant 內操作。

## 2. 硬體與感測器

| 元件 | 用途 |
|---|---|
| ESP32-S3 | 主控(Wi-Fi 2.4GHz) |
| LD2412 24GHz 毫米波雷達 | 人體存在偵測(靜止也偵測得到) |
| PIR ×2 | 紅外線移動偵測(輔助 / 可切換為主感測) |
| BH1750 | 環境光照度(lux) |
| HTU21D | 溫度 / 濕度 |
| 繼電器 ×2 | Relay 1 = 電燈、Relay 2 = 風扇 / 抽風機 |
| 牆面開關輸入 ×2 | 接既有實體開關(燈 / 風扇) |
| 蜂鳴器 | 提示音(可關閉 / 可自訂有人音樂) |
| 狀態指示燈 | 運作狀態(可關閉) |
| 多功能實體按鍵 | 校正 / 重啟 / 重置(見第 7 章) |

## 3. 安裝與接線

> [!CAUTION]
> **AC 110V 高壓側必須由合格電工安裝。** 接線前務必**先斷電**,確認電壓符合機型,完成後再送電。觸電會致命。

- **Relay 1（電燈）/ Relay 2（風扇）**:接到對應負載的火線回路。
- **牆面開關輸入 ×2**:接既有實體開關,送電後仍可手動開關燈 / 風扇。
- **感測器位置**:毫米波 / PIR 朝向使用區域,避免正對冷氣出風口、風扇葉片、會晃動的窗簾(易誤判);光照感測避免被燈具直射。

## 4. 連網與加入 Home Assistant

1. **首次上電**:裝置會開啟一個 Wi-Fi 設定熱點(AP)。用手機連上後會自動跳出設定頁(captive portal),輸入你家 Wi-Fi 帳密。
2. **加入 Home Assistant**:裝置連上你家 Wi-Fi 後,Home Assistant 的 **ESPHome 整合**會自動發現它,點「設定 / Adopt」即可加入。
3. 加入後,所有實體(感測、開關、設定)都出現在 Home Assistant 該裝置頁面。

> [!NOTE]
> 需要 Home Assistant 已安裝 **ESPHome 整合**。本機也內建網頁介面(見第 8 章),不經 Home Assistant 也能看狀態。

## 5. 運作原理

**佔用判斷(Occupancy）**:可在「開燈依據」選 **毫米波**(預設,靜止也算有人)或 **紅外線**(移動才算)。

| 情境 | 動作 |
|---|---|
| 偵測到人 + 環境亮度低於門檻 + 非手動模式 | 自動**開燈**(並播放「有人音樂」,可關) |
| 偵測到人 + 非手動模式 | 自動**開風扇**,並啟動延遲關倒數 |
| 人離開,超過「延遲關時間」 | 自動**關燈 / 關風扇** |
| 濕度 > 「開始除濕」門檻 | 自動**開風扇**排濕 |
| 濕度降到（開始除濕 − 下降關閉）以下 | 自動**關風扇** |
| 開啟「手動模式」 | 暫停所有自動邏輯,只靠手動 / 牆面開關 |

> [!TIP]
> 想完全手動控制時,打開 **手動模式 (Manual Mode)**;自動邏輯會全部讓位給你的手動操作與牆面開關。

## 6. 常用設定

加入 Home Assistant 後,在裝置頁可調整(常用項目):

| 設定 | 說明 |
|---|---|
| 開燈依據 | 毫米波 / 紅外線 二選一 |
| 光照低開燈 (Light Threshold) | 低於此 lux 才會自動開燈 |
| 延遲關時間 (Time Out) | 人離開後多久才關(秒) |
| 抽風機延遲關閉 (Fan Timeout) | 風扇延遲關(分鐘:1 / 5 / 10 / 20) |
| 以上開始除濕 / 下降關閉除濕 | 自動除濕的開 / 關濕度門檻 |
| 手動模式 (Manual Mode) | 暫停自動邏輯 |
| 電燈解耦 / 風扇解耦 (Decouple) | 讓牆面開關與自動邏輯互不干涉 |
| 溫度偏移 / 濕度偏移 | 校正感測讀值 |
| 關閉蜂鳴器 / 測試蜂鳴器 | 靜音 / 試音 |
| 有人時音樂 (Presence On) | 自訂偵測到人時的提示鈴聲(RTTTL) |
| 關閉指示燈 | 關掉狀態 LED |

> [!NOTE]
> 進階使用者 / 安裝人員可在 Home Assistant 內進一步微調毫米波雷達的偵測距離與靈敏度,以及執行「動態背景校正」降低誤判(見第 7 章)。

## 7. 實體按鍵與指示燈

**多功能按鍵**(單一實體鍵):

| 操作 | 結果 |
|---|---|
| 短按 | 提示音;**持續按住約 10 秒**會進入毫米波**動態背景校正**(校正期間請離開偵測區) |
| 長按 **5 秒** | 裝置**重新啟動**(兩短嗶聲) |
| 長按 **10 秒** | **恢復原廠設定**(上升音階,2 秒後執行) |

**狀態指示燈**:顯示運作狀態;可在設定內關閉(「關閉指示燈」)。

> [!WARNING]
> 長按 10 秒 = 工廠重置,會清掉 Wi-Fi 與所有自訂設定,需重新配網。請勿誤觸。

## 8. Web UI 與韌體更新

- **本機網頁**:裝置取得 IP 後,於同網段瀏覽器輸入其 IP(可在 Home Assistant 或路由器查到)即可看到內建網頁,檢視即時狀態。
- **韌體更新 (OTA)**:透過 Home Assistant 的 ESPHome 介面、或本機網頁的更新功能進行;更新過程請勿斷電。

---

## 9. 工廠重置

兩種方式:
1. **實體按鍵**:長按 **10 秒**(見第 7 章)。
2. **Home Assistant**:裝置頁的 **Device Reset 重置** 按鈕。

重置會清除 Wi-Fi 連線與所有自訂設定,裝置回到首次配網狀態。

## 10. 故障排除

| 現象 | 處理 |
|---|---|
| 找不到設定熱點 / 連不上 Wi-Fi | 長按 5 秒重啟;仍不行則長按 10 秒重置後重新配網(確認是 2.4GHz Wi-Fi) |
| 沒人卻一直顯示有人 | 毫米波對風扇 / 窗簾 / 出風口會誤判;調整安裝角度,或執行動態背景校正(第 7 章) |
| 夠暗卻不自動開燈 | 調高「光照低開燈」門檻;確認非「手動模式」 |
| 一直不關燈 / 不關風扇 | 縮短「延遲關時間 / 抽風機延遲」;確認非手動模式 |
| 風扇自己一直轉 | 多半是自動除濕在作動(濕度高);調整除濕門檻 |
| 牆面開關沒反應 / 與自動打架 | 視需求開 / 關對應「解耦」設定 |
| 太吵 | 打開「關閉蜂鳴器」 |

## 11. 安全使用

- AC 高壓側請由**合格電工**安裝;接線前**先斷電**。
- 僅供**室內乾燥處**控制額定範圍內的燈具 / 風扇負載,勿超載。
- 請勿自行拆改內部電路;模組含無線電路,改裝可能違反電信法規。
- 浴室 / 潮濕環境安裝時,本體請置於防潮 / 不直接淋水處。
