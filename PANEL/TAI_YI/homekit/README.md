<h2 style="color:#1c3d5a;border-bottom:3px solid #ff6f48;padding-bottom:8px;margin-top:48px;font-size:28px">3. 初次設定(連 Wi-Fi + 配對)</h2>

### 3.1 讓面板連上家中 Wi-Fi

首次開機(或重置後)面板會自動開啟一個設定熱點:

1. 手機 / 電腦的 Wi-Fi 清單找到 **`Automate-Taiyi-XXXXXX`**(XXXXXX 為序號),連上(無密碼)。
2. 系統通常會自動彈出設定頁;若沒有,手動用瀏覽器開 **`http://192.168.4.1/`**。
3. 頁面會列出附近 Wi-Fi → 選你家的 **2.4 GHz** 網路 → 輸入密碼 → **儲存**。
4. 面板自動重啟並連上你的 Wi-Fi。

<blockquote style="border-left:4px solid #007AFF;background:#eff7ff;padding:14px 18px;margin:16px 0;border-radius:0 12px 12px 0;color:#1c3d5a">
💡 務必選 <b>2.4 GHz</b> 網路,HomeKit 裝置不支援 5 GHz。若路由器 2.4/5 GHz 同名,建議暫時分開命名。
</blockquote>

### 3.2 加入 Apple 家庭

1. 開 iPhone / iPad 的**「家庭」App** → 右上「＋」→「加入配件」。
2. 掃描**面板螢幕上顯示的 HomeKit QR Code**(或手動輸入 `111-88-999`)。
3. 依指示完成,裝置會依機型出現對應的磚塊。

> 需要家中有 **Home Hub**(HomePod / Apple TV / 常駐 iPad)才能遠端控制與自動化。

<h2 style="color:#1c3d5a;border-bottom:3px solid #ff6f48;padding-bottom:8px;margin-top:48px;font-size:28px">4. Apple Home 控制</h2>

### 4.1 💧 除濕機

| 元件 | 功能 |
|---|---|
| 開 / 關 | 除濕機電源 |
| 目標濕度 | **30–80%** 可調 |
| 目前濕度 | 面板感測器回報(未配感測器時顯示 `--`)|
| 風速 | **3 段**(弱 / 中 / 強),滑桿會量化到最近段位 |

> 純除濕裝置,Apple Home 不會出現「加濕」選項(預期行為)。

### 4.2 ❄️ 冷氣機

| 元件 | 功能 |
|---|---|
| 模式 | **冷氣 / 暖氣 / 自動**(自動 = 冷氣 25 °C + 風速自動)|
| 目標溫度 | 冷氣 **5–40 °C**、暖氣 **25–40 °C** |
| 目前溫度 | 家電回報之室溫 |
| 風速 | 低 / 中 / 高(「自動」風速請於面板設定)|

### 4.3 🌬️ 全熱交換

Apple Home 上以 **5 個插座開關**呈現,點亮一個即切到該換氣模式(互斥,一次一種):

| 插座 | 模式 | 說明 |
|---|---|---|
| 智慧 | 機器自動調節新風 / 排風 |
| 手動 | 新風、排風檔位自訂 |
| 送風 | 僅送新風 |
| 排風 | 僅排室內空氣 |
| 睡眠 | 低噪音夜間換氣 |

> 新風 / 排風**檔位(1–4)**、**殺菌燈**與**定時關機**請於面板螢幕或 Web UI 調整。

<h2 style="color:#1c3d5a;border-bottom:3px solid #ff6f48;padding-bottom:8px;margin-top:48px;font-size:28px">6. 模式與功能</h2>

### 6.1 除濕機

- **目標濕度** 30–80%,面板感測器閉迴路控制(達標自動停壓縮機)
- **3 段風速**:弱 / 中 / 強
- **定時關機**:面板端倒數,15 檔

### 6.2 冷氣機

| 模式 | 用途 |
|---|---|
| **冷氣** | 製冷,目標 5–40 °C |
| **暖氣** | 製熱,目標 25–40 °C |
| **送風** | 只吹風不調溫 |
| **自動**(Apple Home)| 一鍵冷氣 25 °C + 風速自動 |

- **風速**:自動 / 低 / 中 / 高
- **定時關機**:面板端倒數,15 檔

### 6.3 全熱交換

- **5 種換氣模式**:智慧 / 手動 / 送風 / 排風 / 睡眠
- **新風檔 / 排風檔**:各 1–4 檔(手動模式可獨立調)
- **殺菌燈**:與主機連動,主機關機時自動關閉
- **定時關機**:面板端倒數,15 檔

<h2 style="color:#1c3d5a;border-bottom:3px solid #ff6f48;padding-bottom:8px;margin-top:48px;font-size:28px">9. 故障排除</h2>

### 9.1 配對相關

| 現象 | 原因 | 處理 |
|---|---|---|
| 家庭 App 找不到裝置 | Wi-Fi 是 5 GHz / 距離太遠 | 走 2.4 GHz、靠近路由器 |
| 顯示「無法加入配件」 | 無 Home Hub / 配對碼錯 | 確認家中有 HomePod / Apple TV / 常駐 iPad;重掃面板 QR |
| 配對後顯示「無回應」 | 不同 SSID / mDNS 問題 | 手機與面板同一個 2.4 GHz SSID,關「訪客隔離」 |

### 9.2 運作相關

| 現象 | 原因 | 處理 |
|---|---|---|
| Apple Home 顯示離線 | Wi-Fi 斷線 / 路由器重啟 | 等 1–2 分鐘自動回連 |
| 面板操作但家電沒反應 | RS-485 通訊異常 / 家電關機 | 確認家電電源、RS-485 接線(A/B 是否接反)|
| 濕度 / 室溫顯示 `--` | 未配感測器 / 家電未回報 | 除濕濕度需面板感測器;剛開機第一輪輪詢前也會 `--` |

### 9.3 OTA 相關

| 現象 | 原因 | 處理 |
|---|---|---|
| 版更顯示「無法取得版本」 | 連不到雲端 | 檢查網路 / DNS |
| 「開始更新」卡住 | 下載中斷 | 等 5 分鐘;不行重啟再試 |
