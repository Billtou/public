## 1. 產品介紹

**米多力除濕機智能面板** 是一片 4 吋電容觸控面板，專為 **米多力／貫捷吊隱式除濕機（IN-D12 / IN-D17）** 設計，
以 **Apple HomeKit** 標準接入 **Apple 家庭（Home）**，讓您既能在牆上面板直接操作，也能用 iPhone / iPad、Siri 語音與自動化情境遠端控制。

一片面板同時做兩件事：

- **就地觸控**：牆面面板即時操作除濕、清淨、換氣、風速、目標濕度，反應快、不依賴網路。
- **HomeKit 遙控**：以 Apple HomeKit 接入家庭 App，可遠端控制、Siri 語音、加入自動化與「離家 / 回家」情境。

> **原生 HomeKit** — 面板以 Apple HomeKit 標準自我介紹，**無需安裝專屬 App**，加入 Apple 家庭即可使用。

| 控制方式 | 介面 | 通訊協定 |
| --- | --- | --- |
| 就地直接觸控 | 4" 480×480 電容觸控 LCD | — |
| 手機 / 語音遙控 | Apple 家庭（Home）· Siri | HomeKit over Wi-Fi |
| 連接除濕機主機 | 米多力／貫捷 吊隱除濕機 | RS485（55 通訊協議）|

## 5. 面板功能介紹

面板底部可切換三個分頁：**主控頁**、**設定頁（齒輪）**、**使用說明頁（QR）**。

### 5.1 主控頁 — 除濕控制

| 區域 | 功能 | 說明 |
| --- | --- | --- |
| **左側濕度直條** | 目前濕度 + 目標濕度 | 目前濕度即時顯示；拖曳可調目標濕度 |
| **三顆預設鈕** | 舒爽 / 乾衣 / 連續 | 舒爽=60%、乾衣=50%、連續=不設限持續除濕 |
| **室溫顯示** | 頂部「##°C」 | 依溫度自動變色（藍→綠→橘→紅）|
| **電源卡片**（右上）| 開 / 關 + 模式名 | 顯示「除濕 / 清淨 / 除濕+清淨 / 送風」與自動關機倒數；點按開關機 |
| **四格模式輪**（右中）| 除濕 · 清淨 · 除濕+清淨 · 送風 | 四扇形點選；**再點目前模式一次即關機**；中央為 UV 燈提示（選購）|
| **風速三鍵**（右下）| 弱 · 中 · 強 | 點選切換風速 |
| **壓縮機 / 除霜圖示** | 運轉指示 | 壓縮機運轉、除霜中時顯示對應圖示 |

### 5.2 設定頁（齒輪）

- **背光亮度**（5–100%）與 **螢幕休眠**（10–61 秒或恆亮）
- **連網狀態**（Wi-Fi 訊號 % 或「離線」）
- **濾網時數**「X / Y h」+ 重置鈕；可選門檻（1500 / 2160 / 3000 / 4000 小時）
- **運轉狀態**：正常，或水滿 / 高壓 / 低壓 / 濕度異常等提示
- **UV 燈**開關（選購機種）
- **螢幕重新整理**（面板重繪）

### 5.3 使用說明頁（QR）

四格 QR / 資訊：

| 位置 | 內容 |
| --- | --- |
| 左上 | **HomeKit 配對 QR** + 設定碼 `111-88-999` |
| 右上 | **使用說明 QR**（開啟本說明書最新版）|
| 左下 | **本機網址 QR**（連面板 Web 介面；未連網顯示「未連線」）|
| 右下 | 橘色 **「長按 10 秒 重置連網」** ＋ 綠色 **「重開機」** |

## 6. 接入 Apple 家庭

本面板為 **原生 Apple HomeKit 配件**，直接加入 **Apple 家庭（Home）** App：

1. 開 iPhone / iPad 的 **「家庭」App**
2. 點 **「+」→ 加入配件**
3. **掃面板「使用說明」頁的 HomeKit QR Code**（或手動輸入設定碼 `111-88-999`）
4. 依指示設定 Wi-Fi（2.4 GHz）與命名
5. 完成後配件顯示為 **「米多力除濕機」**

### 加入後，家庭 App 內會出現這些控制

| 配件 / 服務 | 可做什麼 |
| --- | --- |
| **米多力除濕機**（除濕機）| 開 / 關、設定 **目標濕度**、調 **風速**、看目前濕度 |
| **清淨**（空氣清淨）| 開 / 關空氣清淨、調風速 |
| **室溫**（溫度感測）| 檢視目前室溫（唯讀）|
| **換氣**（送風）| 開 / 關送風換氣、調風速 |
| **漏水**（漏水感測）| **水滿 / 漏水時觸發 Apple 家庭緊急警示 + 推播通知** |

> **模式互斥**：除濕 / 清淨 / 送風為互斥運轉；在 App 開啟某一模式時，面板與機器會自動切換。
> 風速三檔（弱 / 中 / 強）在 App 以百分比呈現（約 33% / 66% / 100%）。

### 進階：其他 HomeKit App / Home Assistant

- 可用任何 **HomeKit 相容 App**（如 Eve、Home+、Controller for HomeKit）控制。
- 進階使用者可透過 **Home Assistant 的「HomeKit 控制器（HomeKit Controller）」** 整合接入。
- ⚠️ HomeKit 配件**不支援** Google Home / Amazon Alexa 直接接入（那需要 Matter 版本）。

## 7. Wi-Fi 設定

### 7.1 隨 HomeKit 配對自動設定（建議）

用 **Apple 家庭 App 掃 QR 配對** 時，Wi-Fi 會由家庭 App 自動帶入，**不需手動設定**。

### 7.2 手動設定（SoftAP 救援模式）

未設定 Wi-Fi 的面板會自動開啟設定熱點：

| 項目 | 值 |
| --- | --- |
| 熱點 SSID | `Automate-Medole-XXXXXX`（末 6 碼為 MAC）|
| 密碼 | 無（open）|
| 設定頁 URL | `http://192.168.4.1/`（連上後手機通常自動彈出）|

連上熱點後在設定頁 **選您的家用 Wi-Fi（2.4 GHz）→ 輸入密碼 → 儲存**，面板會自動重啟並連上家用網路。

## 8. 運轉與提示

### 8.1 除濕模式與目標濕度

| 項目 | 說明 |
| --- | --- |
| **目標濕度** | 30–90%；機器依實際濕度自動起停壓縮機 |
| **連續** | 目標設為「連續」= 不設限、持續除濕 |
| **舒爽 60% / 乾衣 50%** | 一鍵預設；乾衣適合陰雨天晾衣、舒爽為日常體感 |
| **除濕 / 清淨 / 除濕+清淨 / 送風** | 四種運轉模式（互斥）|

### 8.2 提示與警示

| 狀況 | 面板顯示 | Apple 家庭 |
| --- | --- | --- |
| **水滿 / 漏水** | 全螢幕紅色「嚴重故障 · 漏水 · 停止運轉」，排水後自動解除 | 「漏水」觸發**緊急警示 + 推播** |
| **除霜中** | 溫度旁雪花圖示 | 除濕暫歇（屬正常保護）|
| **濾網到期** | 設定頁「濾網 X/Y h」轉紅 | — |
| **感測 / 壓力異常** | 設定頁狀態列顯示（水滿 / 高壓 / 低壓 / 濕度異常）| 配件可能顯示閒置 / 無回應 |

> 除濕運轉會在達到目標濕度後暫停壓縮機、濕度回升再啟動，屬正常節能行為。

## 9. 韌體更新 (OTA)

### 9.1 自動檢查

面板啟動後會背景定期檢查
[Billtou/public/PANEL/MEDOLE/homekit/firmware-w](https://github.com/Billtou/public/tree/main/PANEL/MEDOLE/homekit/firmware-w)
上的 `version.json`，若雲端版本較新，**面板 Web 介面**會顯示「有新版可更新」。

### 9.2 手動更新

1. 進 `http://<面板IP>/update`
2. 顯示 **目前版本** vs **最新版本**
3. 點選 **「開始更新」**
4. 韌體下載 → 寫入 → 自動重啟
5. **裝置設定 / HomeKit 配對 / Wi-Fi 設定全部保留**

> ⚠️ 更新中**請勿斷電**。雙分區設計會在異常時自動回到上一版。

## 10. 重新配對

### 10.1 何時需要

- 面板轉送、轉賣 → **必須**重新配對（清除上一手的家庭與 Wi-Fi）
- 更換家用 Wi-Fi 名稱 / 密碼
- 家庭 App 內長時間顯示「無回應」

### 10.2 操作

面板 → **「使用說明」頁 → 右下橘色「長按 10 秒 重置連網」**，按住滿 10 秒（畫面會閃爍提示）即執行：

- ✅ 清除 HomeKit 配對
- ✅ 清除 Wi-Fi 設定 → 重新進入設定熱點
- ❌ 韌體版本不變
- ❌ 製造日期 / 序號不變

> 重新配對後，請先在 Apple 家庭 App 內移除舊配件，再重新加入。

## 11. 故障排除

<details>
<summary><b>面板無畫面 / 不亮</b></summary>

1. 確認 DC 5V 電源接頭已插緊
2. 拔電源 10 秒後重插
3. 仍無反應 → 聯絡售後

</details>

<details>
<summary><b>面板正常但顯示除濕機「離線」</b></summary>

1. 確認 RS485 接線正確、端子鎖緊
2. 確認除濕機主機已通電
3. 拔 RS485 線與電源 → 重新接妥 → 再上電（避免熱插拔暫態）

</details>

<details>
<summary><b>HomeKit 配對失敗</b></summary>

1. 手機與面板需在**同一 Wi-Fi（2.4 GHz）**
2. 確認 iPhone / iPad 藍牙已開（配對輔助用）
3. 路由器需允許 mDNS / Bonjour（預設都允許）
4. 部分雙頻路由器會把手機切到 5 GHz，請確認連的是 2.4 GHz
5. 曾配對過 → 先用面板「長按 10 秒 重置連網」清除，再重配

</details>

<details>
<summary><b>家庭 App 顯示「無回應」但面板觸控正常</b></summary>

1. 通常是路由器 mDNS 暫時失靈 → 重啟路由器
2. 確認面板 Wi-Fi 沒掉線（設定頁看訊號）
3. 仍無回應 → 重新配對

</details>

<details>
<summary><b>一直顯示「漏水 / 水滿」</b></summary>

1. 檢查除濕機水箱 / 排水管是否已滿或阻塞
2. 排水後提示會自動解除
3. 若已排水仍持續 → 檢查機器水位感測，聯絡售後

</details>

<details>
<summary><b>濾網提醒一直亮</b></summary>

1. 清潔或更換濾網後，到 **設定頁 → 濾網 → 重置** 歸零時數

</details>

<details>
<summary><b>OTA 更新卡住</b></summary>

1. **請勿斷電**，等候數分鐘
2. 仍卡住 → 拔電重開，雙分區設計會自動回上一版
3. 重試；若仍失敗 → 檢查 Wi-Fi 連線品質

</details>

## 12. 安全使用

- ⚡ 僅使用隨附的 DC 5V 電源，請勿替換不明規格變壓器。
- 🧱 面板嵌入標準歐式接線盒，安裝 / 移除前請先**斷開電源**。
- 💧 面板為室內牆面安裝，請勿直接噴水沖洗；清潔時關電源、用微濕軟布擦拭。
- 🌡 工作環境 0–50 ℃、濕度 10–85% 無凝露。
- 🔌 RS485 接線與除濕機安裝建議由**專業安裝商**施工，並依除濕機本體說明書操作。
- 🚱 「水滿 / 漏水」為安全保護，請依提示排水後再使用。
