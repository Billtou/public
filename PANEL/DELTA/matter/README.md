## 1. 產品介紹

**DeltaPanel** 是一片 4 吋電容觸控面板，專為 **台達衛浴與通風三件套**——
**中央暖風機（浴霸）**、**全熱交換器（ERV）**、**管道送風機（新風）**——所設計的智能控制中樞。

一片面板同時做兩件事：

- **就地觸控**：在牆上面板直接操作主機所有功能，反應即時、不依賴網路。
- **智能遙控**：以 **Matter over Wi-Fi** 標準接入 Apple Home / Google Home / Home Assistant，
  讓您在手機、語音助理或自動化情境中遠端控制。

> **一鍵接入**：面板以 Matter 標準向智能 App 自我介紹，**無需安裝專屬 App**，
> 也能同時被多個智能平台共享（Matter Multi-Admin）。

<div align="center">

| 控制方式 | 介面 | 通訊協定 |
| --- | --- | --- |
| 就地直接觸控 | 4" 480×480 電容觸控 LCD | — |
| 手機 / 語音遙控 | Apple Home / Google Home / HA | Matter over Wi-Fi |
| 連接台達主機 | 暖風機 / 全熱交換器 / 管道風機 | Modbus RTU (RS485) |

</div>

## 2. 台達三件套

同一片 DeltaPanel，依安裝時搭配的台達主機，**出廠即設定為對應的控制模式**，
面板畫面與智能 App 內呈現的功能會自動切換成該主機專屬的介面：

<table>
<tr>
<td width="33%" valign="top">

### 🔥 浴霸模式
**中央暖風機**

- 暖風 / 涼風 / 換氣 / 乾燥四大模式
- 主燈、夜燈、韻律風、感應通風
- 室溫顯示、定時關機
- 一鍵大紅停止鍵

## 5. 快速開始

```text
1. 拆封 → 接電源 → LCD 亮起
2. 滑到「使用說明」分頁 → 掃 QR 看完整本說明
3. 開 Apple Home / Google Home / Home Assistant App
4. 加入新設備 → 掃面板上的 Matter QR / 手動輸入配對碼
5. 設定 Wi-Fi（2.4 GHz）→ 等候 30–90 秒接入完成
6. 接入完成後即可在 App 遙控，面板亦可同時就地操作
```

## 6. 面板功能介紹

### 6.0 共通操作

| 區域 | 功能 |
| --- | --- |
| **底部分頁** | 主控頁 ｜ 設定頁 ｜ 使用說明頁（含 Matter QR、本機網址、重置鍵） |
| **設定頁** | 背光亮度 0–100%、濾網時數重置（依機型）、其他偏好 |
| **狀態圖示** | 右下角顯示 Wi-Fi / Matter 連線狀態 |
| **使用說明頁** | 掃 QR 看最新說明書、看 Matter 配對碼、看本機 Web 網址、長按重置 |

> 以下依面板出廠對應的主機，介紹各模式主控頁的實際按鍵與顯示。

### 6.1 🔥 浴霸模式（中央暖風機）

主控頁一眼可操作：

| 功能 | 操作 | 說明 |
| --- | --- | --- |
| **電源** | 左側大卡片點按 | 一鍵開 / 關 |
| **室溫** | 右側顯示 | 即時顯示浴室溫度 |
| **暖風 / 涼風** | 模式磚 | 各分 **弱 / 強** 兩段 |
| **換氣** | 模式磚 | 排出濕熱空氣，分弱 / 強 |
| **乾燥** | 模式磚 | 浴後乾燥，分 **涼乾 / 熱乾** |
| **定時** | 步進鈕 | 0 = 持續運轉，最長可設數小時後自動關機 |
| **主燈 / 夜燈** | 功能磚 | 獨立開關照明與柔光夜燈 |
| **韻律風** | 功能磚 | 自動擺動送風，受熱更均勻 |
| **感應通風** | 功能磚 | 偵測到人自動換氣 |
| **🔴 停止** | 大紅按鈕 | 一鍵全部停止 |

> 底部會顯示 **濾網時數 / 機體運轉時數**，清潔濾網後可在設定頁一鍵歸零。

### 6.2 🌬 全熱模式（全熱交換器 / ERV）

主控頁：

| 功能 | 操作 | 說明 |
| --- | --- | --- |
| **電源** | 左側大卡片點按 | 一鍵開 / 關 |
| **風量預設** | 風量磚 | **低 / 中 / 高** 三段直選 |
| **自訂風量** | 「自X ▼」磚 | 三組自訂（自1 / 自2 / 自3），可下拉切換 |
| **送風百分比** | 滑桿 | 自訂模式下，送風 0–100% 微調 |
| **排風百分比** | 滑桿 | 自訂模式下，排風 0–100% 微調 |
| **外氣 / 回風溫度** | 右側顯示 | 即時溫度監測 |
| **送 / 排風轉速** | 右側顯示 | 即時 RPM |

設定頁（依機型提供）：

- **氣流模式**：熱交換 ／ 旁通 ／ 自動 ／ 內循環
  （旁通、自動、內循環為部分機型專屬，面板會自動偵測主機是否支援並顯示對應選項）
- **濾網時數**：顯示已使用時數，更換濾網後可一鍵重置

> 春秋季可用 **旁通 / 自動** 引入室外涼風，省去熱交換的能耗。

### 6.3 💨 新風模式（管道送風機）

主控頁：

| 功能 | 操作 | 說明 |
| --- | --- | --- |
| **電源** | 左側大卡片點按 | 一鍵開 / 關，並顯示「正在運轉 / 已停止」 |
| **低風量 / 高風量** | 風速磚 | 兩段快選 |
| **多段** | 風速磚 | 1–17 段細緻風量 |
| **段位微調** | 滑桿 | 多段模式下以百分比無段調整 |
| **即時轉速** | 右側顯示 | 大字顯示目前 RPM |

設定頁（依機型提供）：

- **濾網時數**：顯示已使用時數，更換濾網後可一鍵重置

</td>

## 8. Matter 配對

### 8.1 看 Matter 配對碼

面板開機後，點選底部 **「使用說明」** 分頁，可看到：

| 資訊 | 用途 |
| --- | --- |
| **Matter 配對碼**（數字）| Apple Home / Google Home 手動輸入用 |
| **Matter QR Code** | 各 App「掃描加入」用 |
| **使用說明 QR** | 掃描可開啟本說明書最新版 |
| **本機網址 QR** | 掃描可開啟 Web 管理介面 |

> **出廠配對碼為固定值** — 這是 Apple / Google / Matter 規範允許的「公開測試憑證」用法。
> 配對成功後 Wi-Fi 端會啟動加密通道，配對碼僅用於初次握手。

### 8.2 配對前置條件

- ✅ 手機已加入 Wi-Fi（2.4 GHz）且 Bluetooth 開啟
- ✅ DeltaPanel 已通電，LCD 已亮起
- ✅ 路由器允許 mDNS / IPv6（預設都開，除非有特別封鎖）
- ✅ 手機與面板在同一 Wi-Fi 子網

### 8.3 配對中

App 會顯示 **「正在搜尋…」→「設定 Wi-Fi…」→「測試 Matter 連線…」** 三階段。
**全程約 30–90 秒**，過程中請勿關閉 App 或拔面板電源。

## 10. RS485 接線

### 10.1 接線圖

```text
    ┌──────────────┐         ┌────────────────────┐
    │  DeltaPanel  │   A+    │  台達暖風機 / ERV   │
    │              ├─────────┤   全熱交換器        │
    │  (RS485)     │   B-    │  管道送風機         │
    │              ├─────────┤  (Modbus Slave)    │
    └──────────────┘         └────────────────────┘

    - A+ ↔ A+, B- ↔ B-（切勿交叉）
    - 長度 < 100 m 不需另加終端電阻（面板已內建 120Ω）
    - 多從機並聯時，僅最遠端那台另外掛 120Ω 即可
```

### 10.2 支援的台達從機

| 系列 | 預設位址 | 自動發現 |
| --- | --- | --- |
| FMBG / IAQ 中央暖風機 | `0x01` | ✅ |
| VEB 全熱交換器 (ERV) | `0x64`（100）| ✅ |
| VDB / DPT / DHT 管道風機 | `0xC8`（200）| ✅ |

面板首次開機會自動掃預設位址，**有反應的從機自動上線並出現在 App 中**。

### 10.3 多從機 / 改位址

若您改了從機的 Modbus 位址（廠家設定軟體做），DeltaPanel 必須重新掃描：
**Web 管理介面 → 主頁 → 重新整理（3 秒自動更新）**。

## 13. 重新配對

### 13.1 為什麼要重新配對?

- 把面板轉送、轉賣 → **必須**重新配對（避免上一手帳號殘留）
- 改家用 Wi-Fi 名稱 / 密碼 → 需要重新配對
- App 內顯示「離線」且長時間不恢復 → 重新配對

### 13.2 兩種方式

| 方式 | 操作 | 適用情境 |
| --- | --- | --- |
| **LCD 長按 10 秒** | 面板 →「使用說明」分頁 → 橘色「長按 10 秒 重置連網」按鈕 | 面板可正常觸控 |
| **Web 介面** | `http://<IP>/update` → 紅色「重新配對」卡 | 不在面板旁邊，但能連到網路 |

> 按住期間面板會閃爍提示，**滿 10 秒**才會執行；未滿放開即取消。

### 13.3 重新配對後狀態

- ✅ Matter 配對資訊清除 → 進入待配對狀態
- ✅ Wi-Fi 連線設定清除 → 重新進入 SoftAP 模式
- ❌ 韌體版本不變
- ❌ 製造日期 / 序號不變
- ❌ 面板對應的主機模式不變

## 14. 故障排除

<details>
<summary><b>面板無畫面 / LED 不亮</b></summary>

1. 確認 DC 5V 電源接頭已插緊
2. 拔電源 10 秒後重插
3. 檢查電源轉接器 LED 是否亮（紅 = 正常）
4. 仍無反應 → 聯絡售後

</details>

<details>
<summary><b>面板畫面正常但 RS485 設備離線</b></summary>

1. 確認 A+ / B- 接線正確（顏色標示）
2. 從機端的 Modbus 位址必須是預設值（`0x01` / `0x64` / `0xC8`），若改過位址需用廠家軟體改回
3. Web 主頁等 3 秒，刷新看「RS485 設備」卡是否上線
4. 拔 RS485 線拔電源 → 重新接 → 開電源（避免熱插拔造成的暫態）

</details>

<details>
<summary><b>Matter 配對失敗</b></summary>

1. 手機 Bluetooth 必須開啟（僅初次配對用）
2. 手機與面板必須在**同一 Wi-Fi 子網**
3. 路由器 mDNS / IPv6 必須允許（預設都允許）
4. 嘗試切換手機 Wi-Fi → 確認連的是 2.4 GHz 那個 SSID（部分雙頻路由器會自動切 5 GHz）
5. 重啟路由器 + 面板，重試
6. 若曾配對過，先用 LCD「長按 10 秒 重置連網」清除舊狀態再重配

</details>

<details>
<summary><b>App 內顯示「離線」但面板觸控正常</b></summary>

1. 通常是路由器 mDNS 暫時失靈 → 重啟路由器
2. 確認面板 Wi-Fi 沒掉線（LCD 右下角圖示）
3. 嘗試開 `http://<IP>` 看 Web 介面是否能連到，能連到代表面板正常，問題出在 App 端
4. 仍離線 → 重新配對

</details>

<details>
<summary><b>App 內看不到某個功能（如弱/強段位、送排風百分比）</b></summary>

1. 部分細緻調整（暖風弱/強段位、ERV 送排風百分比微調）設計為**面板就地操作**，不在 App 暴露
2. 請於面板主控頁進行；App 端負責開關、模式與自動化
3. 這是正常設計，非故障

</details>

<details>
<summary><b>OTA 升級卡住</b></summary>

1. **絕對不要拔電源** — 等候 5 分鐘
2. 若 5 分鐘後仍卡住，拔電源重開，雙分區設計會自動回上一版
3. 重試 OTA；若仍卡 → 檢查面板 Wi-Fi 連線品質
4. 仍失敗 → 聯絡售後

</details>

<details>
<summary><b>面板觸控失靈 / 漂移</b></summary>

1. 用乾淨柔軟的布（無水）擦拭面板表面
2. 拔電源 10 秒重開
3. 仍失靈 → 聯絡售後（觸控感測器可能需要校正或更換）

</details>

<details>
<summary><b>濾網提醒一直亮</b></summary>

1. 更換或清潔濾網後，需手動歸零時數
2. 面板 → 設定頁 → 濾網時數重置（依機型提供）
3. 浴霸模式：設定頁可重置風扇 / 濾網運轉時數

</details>

## 15. 安全使用

- ⚡ 僅使用隨附的 DC 5V 電源，請勿替換不明規格的變壓器。
- 🧱 面板嵌入標準歐式接線盒，安裝、移除前請先**斷開電源**。
- 💧 面板為室內牆面安裝，請勿直接噴水沖洗；清潔時關電源、用微濕軟布擦拭。
- 🌡 工作環境 0–50 ℃、濕度 10–85% 無凝露；勿安裝於淋浴噴頭正下方等直接淋水處。
- 🔌 RS485 接線、主機安裝建議由**專業安裝商**施工，並依台達主機本體說明書操作。
- 🚸 暖風機高溫運轉時出風口溫度較高，請留意孩童與寵物。
