# 第一步連接自家WIFI
## - 上電等待1-2分鐘，手機靠近設備找wifi熱點 weatherpanel-auto-xxxxxx，接入後手機切換到瀏覽器輸入192.168.4.1即可指定自家wifi與密碼(wifi 僅支持 2.4G)。若連接成功靜待1-2分鐘，畫面就會自動更新。

  ### 畫面右上方會有設備的ip位置，手機的瀏覽器輸入後會跳出後台管理頁面
