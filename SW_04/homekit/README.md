# AUTOMATE 四路 HomeKit 智慧開關橋接器 (v1.2.0)

歡迎使用 AUTOMATE 四路智慧開關！本產品專為專業智慧家庭設計，透過 ESP32-S3 強大效能，讓您能直接使用 Apple Home (家庭) App 獨立控制 4 路高功率 AC 開關。

## 🚀 產品核心特性

*   **原生 Apple HomeKit 支援**：無需額外網關或橋接器，直接連入 Apple 系統。
*   **雙網路自適應切換**：支援 W5500 乙太網 (有線) 與 2.4GHz Wi-Fi (無線) 自動備援。
*   **4 路獨立控制**：一台設備即可管理四組獨立電路。
*   **優化 LED 狀態顯示**：精確反饋連網與配對狀態，運行時燈號自動熄滅不干擾。
*   **動態 OTA 更新**：透過本地網頁後台即可輕鬆升級最新韌體。

## 🌐 網路切換使用說明 (核心功能)

本設備具備 **「乙太網優先」** 的智慧切換邏輯：

### 1. 乙太網 (有線) 使用模式
*   **使用方法**：直接插上網線後開機。設備偵測到有線連接後會自動將其作為主通道。
*   **優勢**：連線最穩定、反應延遲最低。

### 2. Wi-Fi (無線) 使用模式
*   **切換機制**：若開機時未插網線，或在運行中拔掉網線，設備會自動啟動 Wi-Fi 備援機制並嘗試連接已設定的無線網路。
*   **設定方法**：
    1. 手機連接至設備發出的熱點：`AUTOMATE-SW4-W-XXXXXX`。
    2. 自動彈出設定頁（或訪問 `192.168.4.1`）。
    3. 掃描並選擇家中 Wi-Fi 並輸入密碼，設備重啟後即可進入無線模式。

### 🚨 注意事項與常見問題
*   **優先權機制**：當同時插有網線並設定有 Wi-Fi 時，設備會**優先關閉 Wi-Fi** 以節省效能並確保有線連線品質。
*   **切換延遲**：從 Wi-Fi 切換到乙太網（插上網線）通常在 2-5 秒內完成；從乙太網切換到 Wi-Fi（拔掉網線）可能需要約 10 秒進行重新掃描與連線。
*   **若切換失敗**：如果在插拔網線後設備長時間未恢復連線（燈號持續閃爍），請嘗試斷電重啟。
*   **網路環境**：Wi-Fi 僅支援 2.4GHz 頻段，請確保路由器已開啟此頻段。

## 📱 HomeKit 配對指南

1.  開啟 **Apple Home (家庭)** App。
2.  點擊 **+** -> **加入配件**。
3.  點擊「我沒有代碼或無法掃描」，選擇 **「手動輸入代碼」**。
4.  **輸入配對碼：`111-88-999`**
5.  等待「AUTOMATE-SW4-Bridge」出現並完成設定。

## 💡 LED 狀態指示燈

| 燈號狀態 | 代表含義 |
| :--- | :--- |
| **慢閃 (1Hz)** | 尚未連網（等待網線接入或 Wi-Fi 設定） |
| **雙閃 (兩快一滅)** | 已連上網路，但**尚未配對**至 HomeKit |
| **熄滅 (OFF)** | **正常運作中**（已連網且已完成 HomeKit 配對） |
| **快閃 (10Hz)** | 重置模式（長按 Reset 鍵中） |

---

## 🔄 設備重置與恢復出廠

1.  找到 **Reset 按鍵** 。
2.  持續按住 **5 秒以上**，直到 LED 開始極速快閃，等到停止快閃後鬆開按鍵。
3.  鬆開按鍵，設備會清除 Wi-Fi 憑據與 HomeKit 配對資訊並自動重啟。
